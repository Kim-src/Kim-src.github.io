### 📌 HTML 소개
> - HTML은 HyperText Markup Language의 약자입니다.
> - 웹 페이지의 구조를 정의하는 데 사용되는 주요 언어입니다.
> - HTML은 태그(tag)와 요소(element)로 구분됩니다.
> - 태그는 ```<head>, <body>, <footer>``` 등을 지칭합니다.
> - 요소는 태그 내부의 type, id, class, name 등을 의미합니다.
